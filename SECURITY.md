## Report a security issue

Report security issues privately through [GitHub's security advisory form](https://github.com/tago-io/snippets/security/advisories/new). Do not report security vulnerabilities on the public issue tracker.

## Vulnerability coordination

Confirmed vulnerabilities are coordinated through the [GitHub Security Advisory](https://docs.github.com/en/code-security/security-advisories) opened for the issue. Affected parties include the reporter, users of TagoIO Snippets, and downstream maintainers where relevant.

## Security advisories

Security fixes are disclosed through [GitHub release notes](https://github.com/tago-io/snippets/releases).
